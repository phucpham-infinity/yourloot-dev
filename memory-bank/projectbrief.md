# Project Brief: TTE Tokyo Budget

## Project Overview

TTE Tokyo Budget is a React-based web application for a gaming/betting platform with wallet management, deposit/withdrawal functionality, and user account features.

## Core Requirements

- User authentication and registration
- Wallet management (fiat and cryptocurrency)
- Deposit and withdrawal functionality
- Bonus and promotion system
- User profile management
- Multi-language support (EN, RU, AZ, KZ, YZ)
- Responsive design with mobile support

## Technical Stack

- React with TypeScript
- Vite as build tool
- Firebase for backend services
- CSS-in-JS with emotion
- Redux for state management
- React Router for navigation
- Internationalization (i18n)

## Key Features

- Form building system with reusable components
- UI component library with custom styling
- Authentication guards and routing
- API service layer with interceptors
- Responsive design patterns
- Dark theme with gradient backgrounds

## Current Focus

Extracting reusable components to improve code organization and maintainability.
